### 例如：工厂的布局

智能生产既有动态方面的，也有静态方面的，以及各种各样的细微差别。
在工厂的场地上把车间设在哪里是一个相当静态的问题，但却是相当重要的方面。
假设我们拥有一家公司，买了一块地要建一座新工厂。
我们当然知道我们将在这个工厂生产哪些产品。
我们也知道我们需要建造哪些设施，例如，车间，仓库，也许还有行政大楼。
我们需要决定的是将它们安置在我们的土地上，如[@fig:qap_example]所示。

\rel.figure{qap_example}{二次指派问题场景的草图，其中不同的工厂建筑需要在一块土地上布局。}{qap_example.svgz}{width=72%}

假设在这片土地上有$n$个位置可以放置$n$个设施。
在一些地方，可能已经有建筑了，在另一些地方，我们可能需要建造新的建筑。
对于每个可能的设施和位置组合，可能会产生不同的建设成本。
一个现有棚子的位置可能是放置仓库的一个好解决方案。
然而，如果我们想把行政大楼放在那里，我们首先需要拆除棚子。 

对于每一个可能的计划，我们希望放置的设施之间的相对距离也产生了成本。
也许两个车间之间有大量的物料流动。
成品和原料可能需要在车间和仓库之间运输。
另一方面，在行政大楼和车间之间，通常没有物质流动。
当然，两个设施之间的距离取决于我们为它们选择的位置。   
对于我们在地图上放置的每一对设施，流动成本将作为它们之间运输的材料数量和它们所在位置之间的距离的数学函数出现。

因此，将设施分配到地点的总费用是由此产生的基本费用和流动费用的加权总和。
我们的目标是找到总成本最小的任务。

这种情况称为二次分配问题（Quadratic Assignment Problem, QAP）[@BCPP1998TQAP]。
二次分配问题是20世纪50年代以来的一个研究课题[@BK1957APATLOEA]。
QAPs 二次分配问题出现在各种各样的场景中，例如设施在一块土地上的位置或工厂车间的工作站的位置。
即使我们需要在电路板上放置元件，以使电线的总长度最小，我们基本上也有一个二次分配问题[@S1961TBWPAPA]！
尽管二次分配问题相对容易理解，但却很难解决[@SGA1976PCAP]。
